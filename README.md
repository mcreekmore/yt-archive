### yt-dl-sync

## Install

```bash
pip3 install yt_dlp dotenv

chmox +x yt_dl-sync.py
```

Adding to cron job is available as a Ansible playbook in the [pms](https://github.com/mcreekmore/pms) repo

## Run Manually

```bash
./archive.py
```
